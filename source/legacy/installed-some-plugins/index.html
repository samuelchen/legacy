>Installed Some Plugins

2006-03-14
未分类

准备了一些插件，清单如下（粗体是安装了的）:

  • quoter: 注释一对一回复引用.
  • dropcap: 首字下沉.
  • af-extended-live-archive: 使用Ajax技术的存档浏览插件.
  • zh_CN: Wordpress 2.01 中文包.
  • iG_Syntax_Hiliter: 代码高亮插件.
  • Image Headlines: 文字变成图像标题. 下载
  • mulberrykit: 显示随机文章，最新留言，留言最多文章，发表评论最多的网友，以及真正的文章摘要，等等，真正截断，没有乱码。这个插件由 WordPress 随机文章和 WordPress 评论插件合并增强而来。在激活这个插件之前，请务必先停用这两个插件，不然的话会有冲突。
  • gengo: 多语言插件.
  • def-link: 对定义过的关键字自动加连接或者注释。
  • gravatar: 评论中添加个人的小图像(未下载).
  • adsense-deluxe: google adsense 管理显示工具。
  • wp-cron: 定时任务插件，可以配合备份使用.
  • ad-rotator: 从文本文件中读取随即广告代码并随机显示，可限制数量，可用于其他系统。
